# neovim config

## Install

把 neovim 作为 IDE 的配置。基于 lua.

Linux:
git clone neovim-config ~/.config/nvim

Windows:
git clone https://gitee.com/aczz/neovim-config $env:USERPROFILE/AppData/Local/nvim

## trouble shooting

rm -rf ~/.local/share/nvim/
